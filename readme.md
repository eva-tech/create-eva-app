# Usage

<!-- usage -->

```sh-session
$ npm install -g create-eva-app
$ create-eva-app COMMAND
running command...
$ create-eva-app (-v|--version|version)
create-eva-app/0.0.0 darwin-x64 node-v12.10.0
$ create-eva-app --help [COMMAND]
USAGE
  $ create-eva-app COMMAND
...
```
